# Vim 常用快捷键与命令总结

## 一、模式介绍

Vim 有三种常用模式：

* **普通模式（Normal Mode）**：默认模式，用于移动光标、删除、复制等
* **插入模式（Insert Mode）**：编辑文本
* **命令模式（Command Mode）**：执行保存、退出等操作

---

## 二、模式切换

| 操作    | 说明            |
| ----- | ------------- |
| `i`   | 在光标前进入插入模式    |
| `a`   | 在光标后进入插入模式    |
| `o`   | 在下一行插入并进入插入模式 |
| `Esc` | 返回普通模式        |
| `:`   | 进入命令模式        |

---

## 三、光标移动（普通模式）

### 基本移动

| 快捷键 | 说明 |
| --- | -- |
| `h` | 左  |
| `j` | 下  |
| `k` | 上  |
| `l` | 右  |

### 快速移动

| 快捷键 | 说明        |
| --- | --------- |
| `w` | 跳到下一个单词开头 |
| `b` | 跳到上一个单词   |
| `e` | 跳到单词结尾    |
| `0` | 行首        |
| `^` | 第一个非空字符   |
| `$` | 行尾        |

### 页面移动

| 快捷键        | 说明    |
| ---------- | ----- |
| `gg`       | 文件开头  |
| `G`        | 文件末尾  |
| `Ctrl + d` | 向下翻半页 |
| `Ctrl + u` | 向上翻半页 |

---

## 四、编辑操作

### 删除

| 快捷键  | 说明     |
| ---- | ------ |
| `x`  | 删除当前字符 |
| `dd` | 删除当前行  |
| `dw` | 删除一个单词 |
| `d$` | 删除到行尾  |

### 复制（Yank）

| 快捷键  | 说明     |
| ---- | ------ |
| `yy` | 复制当前行  |
| `yw` | 复制一个单词 |
| `y$` | 复制到行尾  |

### 粘贴

| 快捷键 | 说明   |
| --- | ---- |
| `p` | 在后粘贴 |
| `P` | 在前粘贴 |

---

## 五、撤销与重做

| 快捷键        | 说明 |
| ---------- | -- |
| `u`        | 撤销 |
| `Ctrl + r` | 重做 |

---

## 六、查找与替换

### 查找

| 命令      | 说明   |
| ------- | ---- |
| `/word` | 向下查找 |
| `?word` | 向上查找 |
| `n`     | 下一个  |
| `N`     | 上一个  |

### 替换

```
:s/old/new/g        # 当前行替换
:%s/old/new/g       # 全文替换
:%s/old/new/gc      # 全文替换（带确认）
```

---

## 七、文件操作（命令模式）

| 命令            | 说明        |
| ------------- | --------- |
| `:w`          | 保存        |
| `:q`          | 退出        |
| `:wq`         | 保存并退出     |
| `:q!`         | 强制退出（不保存） |
| `:e filename` | 打开文件      |

---

## 八、可视模式

| 快捷键        | 说明   |
| ---------- | ---- |
| `v`        | 字符选择 |
| `V`        | 行选择  |
| `Ctrl + v` | 块选择  |

常用操作：

* 选中后 `d` 删除
* 选中后 `y` 复制

---

## 九、实用技巧

### 数字 + 命令

```
3dd   # 删除3行
5w    # 向前跳5个单词
```

### 重复操作

```
.     # 重复上一次操作
```

### 快速缩进

```
>>    # 右缩进
<<    # 左缩进
```

---

## 十、总结

* Vim 的核心是 **普通模式操作文本**
* 熟练使用 **组合命令（如 d + w）**
* 多用 **重复操作（.）提升效率**
